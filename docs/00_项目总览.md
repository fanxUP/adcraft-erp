# 00. 项目总览

## 1. 项目背景

广告制作安装公司日常业务通常涉及客户咨询、尺寸测量、报价、设计、制作、外协、安装、验收、收款和长期对账。传统 Excel、微信、纸质单据、个人电脑文件夹很容易造成以下问题：

- 报价版本混乱，客户确认的是哪一版不清楚。
- 设计文件分散在不同设计师电脑里，后期找不到源文件。
- 生产制作进度靠口头沟通，老板不知道哪个单子卡住。
- 安装现场缺少照片、验收和签字记录，后期容易扯皮。
- 客户长期欠款难统计，对账耗时。
- 材料、外协、人工成本没有沉淀，利润不清楚。

本系统用于解决这些实际问题。

## 2. 系统定位

系统定位为：**广告制作安装工程业务管理系统**。

它不是通用进销存，也不是单纯财务系统，而是结合广告行业特点的业务中台。

核心能力：

```text
客户管理 + 报价管理 + 订单管理 + 设计任务 + 制作任务 + 安装派工 + 外协管理 + 收款对账 + 利润报表 + 文件归档
```

## 3. 部署目标

- 部署在公司局域网服务器。
- 员工电脑通过浏览器访问。
- 手机连接公司 Wi-Fi 后可访问移动页面。
- 数据保存在公司本地服务器，不依赖公网云服务。
- 支持本地自动备份。

## 4. 第一版范围

第一版必须完成“业务闭环”，而不是追求所有功能一次到位。

第一版范围：

1. 登录与角色权限。
2. 客户资料管理。
3. 产品、材质、工艺、报价规则配置。
4. 报价单创建、修改、导出。
5. 报价转订单。
6. 订单状态流转。
7. 设计任务管理。
8. 制作任务管理。
9. 安装任务管理。
10. 文件上传与订单归档。
11. 收款登记与欠款统计。
12. 客户对账单。
13. 首页经营驾驶舱。
14. 基础报表。
15. 数据备份。

## 5. 非第一版范围

以下功能可以预留，不建议第一阶段强行完成：

- 微信小程序。
- 客户在线付款。
- 自动识别收款截图。
- AI 智能报价。
- 智能排产。
- 安装路线规划。
- 复杂多仓库库存。
- 完整会计凭证系统。

## 6. 核心业务主线

```text
客户咨询
  ↓
客户建档 / 选择老客户
  ↓
新建报价单
  ↓
客户确认报价
  ↓
报价转正式订单
  ↓
生成设计任务 / 制作任务 / 安装任务 / 外协任务
  ↓
任务执行与文件归档
  ↓
安装验收
  ↓
收款 / 部分收款 / 欠款
  ↓
客户对账
  ↓
利润与经营报表
```

## 7. 系统价值

对老板：

- 看清订单金额、回款、欠款、利润。
- 及时发现逾期订单和逾期收款。
- 统计客户、业务员、产品、外协的经营表现。

对业务员：

- 快速报价。
- 快速查看订单进度。
- 跟进客户回款。

对设计师：

- 明确设计任务和截止时间。
- 统一上传设计稿和源文件。
- 保留客户修改意见。

对制作人员：

- 清楚看到尺寸、材质、数量、工艺。
- 避免口头传达导致做错。

对安装人员：

- 手机查看地址、联系人、安装要求。
- 上传现场照片和验收记录。

对财务：

- 清楚每个客户应收、已收、未收。
- 快速生成客户对账单。
