# 09. 测试验收清单

## 1. 登录权限

- [ ] 管理员可以登录。
- [ ] 普通用户不能访问无权限页面。
- [ ] 退出登录后不能继续访问接口。
- [ ] 修改密码后旧密码失效。

## 2. 客户管理

- [ ] 可以新增客户。
- [ ] 可以编辑客户。
- [ ] 可以搜索客户。
- [ ] 客户详情显示报价、订单、收款、欠款。
- [ ] 客户禁用后不能创建新订单。

## 3. 产品/材质/工艺

- [ ] 可以新增产品分类。
- [ ] 可以新增产品。
- [ ] 可以新增材质。
- [ ] 可以新增工艺。
- [ ] 报价页面可以选择产品、材质、工艺。

## 4. 报价管理

- [ ] 可以创建报价单。
- [ ] 可以添加多条报价明细。
- [ ] 输入长宽数量后可以自动计算面积和金额。
- [ ] 支持安装费、设计费、运输费、优惠金额。
- [ ] 报价总金额计算正确。
- [ ] 修改报价后生成版本记录。
- [ ] 报价可以导出或打印。
- [ ] 报价可以转订单。

## 5. 订单管理

- [ ] 报价转订单后生成订单编号。
- [ ] 订单保留报价快照。
- [ ] 订单状态可以流转。
- [ ] 状态变更有日志。
- [ ] 订单详情可以查看任务、文件、收款。

## 6. 设计任务

- [ ] 可以创建设计任务。
- [ ] 可以分配设计师。
- [ ] 设计师可以上传设计稿。
- [ ] 可以记录客户修改意见。
- [ ] 可以标记设计确认。

## 7. 制作任务

- [ ] 可以创建制作任务。
- [ ] 可以更新任务状态。
- [ ] 可以记录质检结果。
- [ ] 可以记录返工原因。

## 8. 安装任务

- [ ] 可以创建安装任务。
- [ ] 可以分配安装人员。
- [ ] 手机浏览器可以打开安装任务页面。
- [ ] 可以上传安装前照片。
- [ ] 可以上传安装中照片。
- [ ] 可以上传安装后照片。
- [ ] 可以提交验收记录。
- [ ] 可以标记返工。

## 9. 文件资料

- [ ] 可以上传图片。
- [ ] 可以上传 PDF。
- [ ] 可以上传设计文件。
- [ ] 文件能绑定订单。
- [ ] 文件能绑定任务。
- [ ] 订单详情可以查看所有文件。
- [ ] 文件可以下载。

## 10. 收款与对账

- [ ] 可以给订单登记收款。
- [ ] 支持一张订单多次收款。
- [ ] 订单已收金额自动累计。
- [ ] 订单未收金额自动计算。
- [ ] 客户总欠款自动计算。
- [ ] 可以生成客户对账单。
- [ ] 对账单金额准确。
- [ ] 收款记录不能直接物理删除，只能作废。

## 11. 首页与报表

- [ ] 首页今日订单金额正确。
- [ ] 首页今日收款金额正确。
- [ ] 首页待设计/待制作/待安装数量正确。
- [ ] 欠款排行正确。
- [ ] 销售日报可按日期筛选。
- [ ] 销售月报可按月份筛选。

## 12. 部署与备份

- [ ] Docker Compose 可以一键启动。
- [ ] 前端、后端、数据库、Redis、MinIO 正常运行。
- [ ] 局域网其他电脑可以访问。
- [ ] 自动备份脚本可以生成备份文件。
- [ ] 可以完成一次测试恢复。

## 13. 数据安全

- [ ] 关键操作写入操作日志。
- [ ] 普通用户不能查看财务敏感数据。
- [ ] 非管理员不能恢复备份。
- [ ] 财务记录作废有原因和操作人。
