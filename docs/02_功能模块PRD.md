# 02. 功能模块 PRD

## 1. 首页驾驶舱

### 目标

让老板、业务、财务、生产人员进入系统后立即看到自己关心的信息。

### 核心卡片

- 今日新增订单数
- 今日订单金额
- 今日收款金额
- 本月订单金额
- 本月已收金额
- 本月未收金额
- 待设计任务
- 待制作任务
- 待安装任务
- 逾期订单
- 逾期未收款
- 外协未回货

### 图表

- 最近 7 天订单金额趋势
- 最近 7 天收款趋势
- 客户欠款排行
- 产品销售排行
- 订单状态分布

### 功能

- 新建报价单

- 报价导出 PDF

- 报价作废

### 报价公式类型

1. 按面积报价：长 × 宽 × 数量 × 单价。
2. 按数量报价：数量 × 单价。
3. 按长度报价：长度 × 数量 × 单价。
4. 按字数报价：字数 × 单价。
5. 工程分项报价：材料费 + 加工费 + 安装费 + 运输费 + 管理费 + 利润。
