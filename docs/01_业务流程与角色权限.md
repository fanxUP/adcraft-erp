# 01. 业务流程与角色权限

## 1. 总业务流程

```text
客户询价
  → 建客户档案
  → 创建报价单
  → 报价审批/确认
  → 报价转订单
  → 拆分任务
      → 设计任务
      → 制作任务
      → 外协任务
      → 采购任务
      → 安装任务
  → 完工验收
  → 收款/欠款
  → 对账
  → 利润统计
```

## 2. 报价流程

```text
业务员新建报价
  → 选择客户
  → 添加报价项目
  → 选择产品/材质/工艺
  → 输入尺寸/数量/安装费/设计费/运输费
  → 系统自动计算金额
  → 保存草稿
  → 内部审核，可选
  → 导出 PDF 或打印
  → 客户确认
  → 转订单
```

报价状态：

- 草稿
- 待审核
- 已审核
- 已发送客户
- 客户已确认
- 已转订单
- 已作废

## 3. 订单流程

```text
报价转订单
  → 自动生成订单编号
  → 确认交付日期
  → 指派业务员/设计师/制作负责人/安装负责人
  → 生成任务
  → 跟踪状态
  → 完工验收
  → 收款结算
```

订单状态：

- 待确认
- 已确认
- 待设计
- 设计中
- 待制作
- 制作中
- 待外协
- 外协中
- 待安装
- 安装中
- 待验收
- 已完工
- 待收款
- 部分收款
- 已结清
- 已取消

## 4. 设计任务流程

```text
订单生成设计任务
  → 管理员/业务员分配设计师
  → 设计师查看需求和素材
  → 上传初稿
  → 客户反馈修改
  → 上传修改稿
  → 客户确认最终稿
  → 归档源文件
```

设计任务状态：

- 待分配
- 待设计
- 设计中
- 待客户确认
- 客户修改
- 已确认
- 已归档

## 5. 制作任务流程

```text
订单生成制作任务
  → 制作主管排产
  → 制作人员领取任务
  → 查看尺寸/材质/工艺/文件
  → 开始制作
  → 质检
  → 合格入待安装/待交付
  → 不合格返工
```

制作任务状态：

- 待制作
- 排队中
- 制作中
- 待质检
- 质检通过
- 返工中
- 已完成

## 6. 安装任务流程

```text
订单进入待安装
  → 安装主管派工
  → 安装人员手机查看任务
  → 到达现场
  → 上传安装前照片
  → 开始安装
  → 上传安装中照片
  → 安装完成
  → 上传完成照片
  → 客户签字/验收
  → 完工归档
```

安装任务状态：

- 待派工
- 已派工
- 前往现场
- 安装中
- 待验收
- 已验收
- 需返工
- 已完工

## 7. 收款对账流程

```text
订单完工或确认后
  → 财务登记收款
  → 支持部分收款
  → 系统计算未收款
  → 超期未收款提醒
  → 生成客户对账单
  → 客户确认
  → 结清归档
```

收款状态：

- 未收款
- 部分收款
- 已收款
- 已退款
- 坏账标记

## 8. 角色权限设计

### 老板/管理员

权限：全部数据、全部功能、系统设置、删除/恢复、报表、备份。

### 业务员

权限：客户、报价、订单、自己的业绩、订单进度、客户欠款查看。  
限制：不能删除收款记录，不能修改系统报价基础数据，不能查看全部利润报表。

### 设计师

权限：查看分配给自己的设计任务、上传设计稿、查看相关订单需求和附件。  
限制：不能看订单利润、不能改财务数据。

### 制作主管

权限：制作任务分配、生产状态更新、质检、返工记录。  
限制：不能改报价金额和收款记录。

### 制作人员

权限：查看自己的制作任务、更新制作进度、上传制作照片。  
限制：不能查看客户欠款和利润。

### 安装主管

权限：安装派工、安装日程、安装验收、返工处理。

### 安装人员

权限：手机查看自己的安装任务、上传照片、提交验收。  
限制：不能查看财务数据。

### 财务

权限：收款、支出、发票、客户对账、供应商付款、欠款统计。  
限制：不能改设计源文件和生产任务核心信息。

### 仓库/采购

权限：材料、库存、采购、入库、出库、供应商资料。

## 9. 权限原则

- 财务数据必须留操作日志。
- 收款记录原则上不允许物理删除，只能作废。
- 报价单转订单后，原报价内容应保留版本快照。
- 订单状态变更必须记录操作人和时间。
- 上传文件必须绑定订单或任务。
