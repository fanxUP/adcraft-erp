# 07. 开发阶段计划与验收标准

## 阶段 0：项目初始化

### 目标

建立基础项目结构、开发环境、数据库连接、登录框架。

### 任务

- 创建 monorepo 或 frontend/backend 双目录结构。
- 配置 Docker Compose。
- 创建 PostgreSQL、Redis、MinIO。
- 后端创建 FastAPI 项目。
- 前端创建 Vue 3 项目。
- 实现登录页面。
- 实现 JWT 认证。
- 实现用户、角色、权限基础表。

### 验收

- `docker compose up -d` 可以启动所有服务。
- 浏览器可以打开登录页面。
- 管理员可以登录。
- 登录后可以进入首页空框架。

## 阶段 1：客户、产品、报价闭环

### 目标

实现客户建档和报价单创建。

### 任务

- 客户 CRUD。
- 产品分类 CRUD。
- 产品 CRUD。
- 材质 CRUD。
- 工艺 CRUD。
- 报价规则 CRUD。
- 报价单 CRUD。
- 报价明细自动计算。
- 报价状态流转。
- 报价版本保存。
- 报价导出 PDF，第一版可用浏览器打印替代。

### 验收

- 可以新建客户。
- 可以配置产品、材质、工艺。
- 可以创建报价单。
- 报价明细输入尺寸和数量后自动计算金额。
- 报价单可以保存、修改、查看。
- 报价修改后有版本记录。

## 阶段 2：订单与任务闭环

### 目标

报价确认后转订单，并生成设计、制作、安装任务。

### 任务

- 报价转订单。
- 订单列表。
- 订单详情。
- 订单状态流转。
- 订单状态日志。
- 设计任务 CRUD。
- 制作任务 CRUD。
- 安装任务 CRUD。
- 订单详情页展示任务。

### 验收

- 报价单可以转订单。
- 订单编号自动生成。
- 订单详情可以看到报价明细。
- 可以为订单创建设计任务、制作任务、安装任务。
- 任务状态变化后订单详情同步显示。

## 阶段 3：文件上传与安装闭环

### 目标

实现订单文件归档和安装现场照片上传。

### 任务

- 文件上传接口。
- 文件绑定订单/任务。
- 文件列表和下载。
- 设计稿上传。
- 制作文件上传。
- 安装照片上传。
- 安装验收记录。
- 客户签字字段，第一版可上传签字图片。

### 验收

- 订单可以上传文件。
- 设计任务可以上传设计稿。
- 安装任务可以上传安装前/中/后照片。
- 订单详情可以查看所有相关文件。

## 阶段 4：财务收款与对账

### 目标

实现收款、欠款、客户对账。

### 任务

- 收款记录 CRUD。
- 订单已收/未收自动计算。
- 部分收款。
- 收款凭证上传。
- 客户欠款统计。
- 客户对账单生成。
- 对账单明细。
- 支出记录基础功能。

### 验收

- 订单可以登记多笔收款。
- 系统自动计算订单未收金额。
- 客户详情可查看总欠款。
- 可以按时间范围生成客户对账单。

## 阶段 5：首页驾驶舱与基础报表

### 目标

让老板能看经营数据。

### 任务

- 首页统计接口。
- 今日订单金额。
- 今日收款金额。
- 本月订单金额。
- 本月未收金额。
- 待设计、待制作、待安装数量。
- 逾期订单提醒。
- 欠款排行。
- 销售日报。
- 销售月报。

### 验收

- 首页能显示真实统计数据。
- 客户欠款排行准确。
- 销售日报/月报可筛选日期。

## 阶段 6：外协、库存与利润

### 目标

增强业务管理能力，开始核算成本和利润。

### 任务

- 外协商管理。
- 外协任务。
- 外协付款。
- 材料库存。
- 入库出库。
- 订单成本录入。
- 订单毛利计算。

### 验收

- 订单可以记录外协成本。
- 订单可以记录材料成本。
- 系统可以计算毛利。

## 阶段 7：优化与上线试运行

### 目标

稳定试运行。

### 任务

- 权限细化。
- 操作日志。
- 数据备份脚本。
- 表格筛选优化。
- 手机端安装页面优化。
- Bug 修复。
- 数据导入工具，客户和产品 Excel 导入。

### 验收

- 公司内部至少试运行 2 周。
- 无核心数据丢失。
- 报价、订单、收款主流程稳定。
- 备份和恢复测试通过。
