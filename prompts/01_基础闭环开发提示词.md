# 01. 基础闭环开发提示词

请先实现广告制作安装工程管理系统的第一阶段基础闭环。

## 本阶段目标

完成：登录权限、客户管理、产品材质工艺管理、报价单、报价转订单、订单基础管理。

## 技术要求

- 前端：Vue 3 + TypeScript + Element Plus
- 后端：FastAPI + SQLAlchemy + PostgreSQL
- 使用 Docker Compose 启动
- 后端接口统一 `/api/v1`
- 金额使用 Decimal/numeric

## 必须实现页面

1. 登录页
2. 首页空框架
3. 客户列表
4. 客户详情
5. 产品管理
6. 材质管理
7. 工艺管理
8. 报价列表
9. 报价编辑页
10. 订单列表
11. 订单详情页

## 必须实现后端模块

1. auth
2. users
3. customers
4. products
5. materials
6. processes
7. quotes
8. orders

## 报价计算要求

报价明细支持：

- 产品
- 材质
- 工艺
- 长
- 宽
- 数量
- 单价
- 面积
- 安装费
- 设计费
- 运输费
- 小计

公式：

```text
面积 = 长 × 宽 × 数量
小计 = 面积 × 单价 + 安装费 + 设计费 + 运输费 + 工艺费 + 其他费用
```

报价单总额 = 明细小计合计 - 优惠金额 + 税费。

## 验收标准

- 可以登录。
- 可以新增客户。
- 可以新增产品、材质、工艺。
- 可以创建报价单。
- 报价金额自动计算。
- 报价可以转订单。
- 订单详情能看到报价明细。
